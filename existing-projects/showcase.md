### Layer 2 & Scaling

**Igra Network**
- **Type**: Based ZK Rollup
- **Website**: [igralabs.com](https://igralabs.com)
- **Description**: EVM-compatible based ZK rollup leveraging Kaspa's BlockDAG as decentralized sequencer and settlement layer
- **Features**:
  - Kaspa BlockDAG as decentralized sequencer
  - Optional settlement on Ethereum
  - Synchronous atomic composability of smart contracts
  - MEV resistance
  - Scalable TPS with fast and secure consensus
  - Bitcoin-grade security with modern programmability
- **Use Case**: Next-generation DeFi and payment applications requiring instant risk systems with composable programmability
- **Why It's Notable**: Solves critical Layer 2 limitations by combining Kaspa's scalability with EVM compatibility and cross-chain interoperability

**Kasplex L2**
- **Type**: zkEVM Layer 2
- **Website**: [docs-kasplex.gitbook.io/l2-network](https://docs-kasplex.gitbook.io/l2-network)
- **Description**: Layer 2 scaling solution bringing EVM smart contracts to Kaspa Network
- **Features**:
  - EVM equivalence for Ethereum dApp compatibility
  - Based Rollup design (Kaspa handles sequencing and data availability)
  - DAG integration with GHOSTDAG consensus
  - PLONK-based zk-SNARKs for cryptographic verification
  - Sub-second finality
- **Use Case**: Deploy Ethereum smart contracts on Kaspa with minimal modifications, enabling DeFi and dApps on Kaspa
- **Why It's Notable**: Brings programmability to Kaspa while maintaining its decentralization and security
